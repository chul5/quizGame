## 프로젝트 개요

Python 객체지향 설계를 기반으로 구현한 터미널 퀴즈 게임입니다.  
세계 지리와 상식은 누구나 친숙한 주제로, 국가·수도·지형·인구 등 다양한 카테고리로 문제를 구성할 수 있어 선택했습니다.

- 퀴즈 풀기 / 추가 / 목록 확인 / 최고 점수 기록 제공
- 게임 데이터는 `state.json`에 저장되어 프로그램 재실행 후에도 유지됨

---

## 실행 방법

```bash
git clone https://github.com/chul5/quizGame.git
cd quizGame
python main.py
```

Python 3.x 환경에서 외부 라이브러리 없이 실행 가능합니다.

## 기능 목록

| 번호 | 기능 | 설명 |
|------|------|------|
| 1 | 퀴즈 풀기 | 저장된 퀴즈를 순서대로 출제하고 최종 점수(%) 표시 |

| 3 | 퀴즈 목록 | 등록된 전체 퀴즈 목록 출력 |
| 4 | 점수 확인 | 역대 최고 점수 확인 |
| 5 | 종료 | 프로그램 종료 (데이터 자동 저장) |

## 클래스 설계

### `Quiz` (quiz.py)

퀴즈 한 문제를 표현하는 클래스입니다.

| 속성/메서드 | 설명 |
|-------------|------|
| `question` | 문제 텍스트 |
| `choices` | 선택지 4개 (list) |
| `answer` | 정답 번호 (1~4, int) |
| `display()` | 문제와 선택지를 터미널에 출력 |
| `check(user_answer)` | 사용자 입력과 정답 비교, bool 반환 |

### `QuizGame` (game.py)

게임 전체 흐름을 관리하는 클래스입니다.

| 메서드 | 설명 |
|--------|------|
| `run()` | 메인 루프 — 메뉴 출력 및 입력 처리 |
| `play()` | 퀴즈 진행, 점수 계산 및 최고 점수 갱신 |
| `add_quiz()` | 새 퀴즈 입력 받아 목록에 추가 |
| `show_list()` | 전체 퀴즈 목록 출력 |
| `show_score()` | 최고 점수 출력 |
| `save()` | `state.json`에 현재 상태 저장 |
| `load()` | `state.json` 읽기, 없거나 손상 시 기본 데이터 복구 |

## 데이터 파일 (state.json)

프로그램 루트에 자동 생성되는 JSON 파일입니다.

### 스키마

```json
{
  "quizzes": [
    {
      "question": "문제 텍스트",
      "choices": ["선택지1", "선택지2", "선택지3", "선택지4"],
      "answer": 1
    }
  ],
  "best_score": 80
}
```

### 필드 설명

| 필드 | 타입 | 설명 |
|------|------|------|
| `quizzes` | array | 퀴즈 객체 배열 |
| `quizzes[].question` | string | 퀴즈 문제 |
| `quizzes[].choices` | array | 선택지 4개 |
| `quizzes[].answer` | integer | 정답 번호 (1~4) |
| `best_score` | integer \| null | 최고 점수 (0~100), 미플레이 시 null |

> `best_score`가 `null`인 경우는 한 번도 퀴즈를 풀지 않은 상태입니다.  
> 0점으로 플레이한 경우와 미플레이 상태를 구분하기 위해 `None`으로 초기화합니다.

## 예외 처리

| 상황 | 처리 방식 |
|------|-----------|
| 빈 입력 | 재입력 요청 메시지 출력 |
| 숫자 외 입력 | 재입력 요청 메시지 출력 |
| 범위 외 숫자 | 유효 범위 안내 후 재입력 |
| `Ctrl+C` / `EOF` | 안전하게 루프 탈출 |
| `state.json` 없음 | 기본 퀴즈 5개로 자동 초기화 |
| `state.json` 손상 (JSON 오류, 키 누락) | 기본 데이터로 복구 후 계속 실행 |
| 파일 저장 실패 (`OSError`) | 오류 메시지 출력, 프로그램 계속 실행 |
