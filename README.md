# 🌍 세계 상식 퀴즈 게임

## 프로젝트 개요
터미널에서 동작하는 세계 상식 & 지리 퀴즈 게임입니다.
퀴즈 풀기, 퀴즈 추가, 목록 확인, 최고 점수 기록 기능을 제공하며
게임 데이터는 JSON 파일로 저장되어 프로그램을 재실행해도 유지됩니다.

## 퀴즈 주제 선정 이유
세계 지리와 상식은 누구나 한 번쯤 접해본 친숙한 주제입니다.
국가, 수도, 지형, 인구 등 다양한 카테고리로 퀴즈를 구성할 수 있어
문제 다양성을 확보하기에 적합하다고 판단했습니다.

## 실행 방법
```bash
git clone https://github.com/chul5/quizGame.git
cd quizGame
python main.py
```
## 기능 목록
- 퀴즈 풀기: 저장된 퀴즈를 순서대로 출제하고 결과를 표시
- 퀴즈 추가: 새로운 퀴즈(문제/선택지/정답)를 등록하고 저장
- 퀴즈 목록: 등록된 전체 퀴즈 목록 확인
- 점수 확인: 역대 최고 점수 확인

## 파일 구조
quiz-game/

## 데이터 파일 설명 (state.json)
프로그램 루트에 자동 생성되는 JSON 파일로, 퀴즈 데이터와 최고 점수를 저장합니다.

### 경로
quiz-game/state.json

### 역할
- 프로그램 종료 후에도 퀴즈 목록과 최고 점수를 유지
- 프로그램 실행 시 이 파일을 읽어 이전 상태를 복원
- 파일이 없거나 손상된 경우 기본 퀴즈 데이터로 자동 복구

### 스키마
{
    "quizzes": [
        {
            "question": "퀴즈 문제 텍스트",
            "choices": ["선택지1", "선택지2", "선택지3", "선택지4"],
            "answer": 1
        }
    ],
    "best_score": 0
}

### 필드 설명
| 필드 | 타입 | 설명 |
|------|------|------|
| quizzes | array | 퀴즈 객체 배열 |
| quizzes[].question | string | 퀴즈 문제 |
| quizzes[].choices | array | 선택지 4개 |
| quizzes[].answer | integer | 정답 번호 (1~4) |
| best_score | integer | 역대 최고 점수 (0~100) |
